## 概述

Browserbase MCP服务器通过Model Context Protocol（MCP）为Claude带来强大的浏览器自动化能力。基于[Stagehand](https://docs.stagehand.dev/)构建的该集成方案，可使用自然语言命令实现AI驱动的网页自动化操作。

<Info>
该服务器使Claude能够控制浏览器、导航网站、与网页元素交互并提取数据——所有这些都通过简单的对话式命令完成。
</Info>

## 会话管理

<Info>
Browserbase MCP 服务器支持单会话和多会话架构，以满足不同的自动化需求。
</Info>

<Tabs>
<Tab title="单会话模式">
**传统方案**
- 同一时间仅激活一个浏览器会话
- 适用于基础自动化任务  
- 自动管理会话生命周期
- 理想选择顺序工作流
</Tab>

<Tab title="多会话模式">
**高级并行处理**
- 多个独立的浏览器会话
- 每个会话保持独立状态
- 具备并行执行能力
- 完美适配复杂工作流
</Tab>
</Tabs>

## 快速开始

<Steps>
<Step title="安装 MCP 服务器">
根据需求选择 NPM 安装、远程托管 URL 或本地开发模式。
</Step>

<Step title="配置认证">
在 MCP 配置中设置 Browserbase API 凭证。
从 [Browserbase 控制台](https://www.browserbase.com/overview) 获取 API 密钥。
</Step>

<Step title="开始自动化">
通过 Claude 使用自然语言指令控制浏览器。
</Step>
</Steps>

<Tip>
准备开始？查看我们的[设置指南](/integrations/mcp/setup)获取详细安装说明。
</Tip>
